Angular 2

  • Ramverk för webb, mobile och desktop

Skillnader från Angular

  • TypeScript.
  • Komponentbaserat, inga Controllers och inget Scope.
  • Bättre prestanda.
  • Förbättrad dependency injection. Läs här!

  • Ramverket mer komponentbaserat, behöver inte använda allt och kan använda vissa saker utanför angular.
  • Använder Rx.Js Observable för asynkrona grejjer.
  • ...

Typescript crash-course

  • Superset av Javascript.
  • Utvecklat av Microsoft.
  • Transpilerar till Javascript och gör statisk analys. Producerar Javascript även om analysen visar på fel.
  • Det finns interface men man behöver inte "implementera" dem. Definerar bara en struktur och ett objekt är ett interface om de har samma inre struktur.
  • Det viktigaste (idag) är typannoteringar och decorator.
  • Typer är Javascript-typer (boolean, number, string ...) och Klasser/interface
  • Typer är inte obligatoriska, Any antas om inget annat anges.

Typannoteringar

Syntax är "variablenamn: Type"


						class Greeter {
    greeting: string;
    constructor(message: string) {
        this.greeting = message;
    }
    greet() {
        return "Hello, " + this.greeting;
    }
}

let greeter = new Greeter("world");

Decorators

Lägger till möjlighet till "metaprogrammering" och annotering. Används av Angular exempelvis för att skapa komponenter.


function f() {
	console.log("Evaluated");
	return function (target, propertyKey: string, descriptor: PropertyDescriptor) {
    console.log("Called");
	}
}

class C {
  @f()
  method() {}
}
